# 📈 Stock Market Prediction using Ensemble Learning & Deep Learning

## 🚀 Overview

This project presents an advanced **stock market prediction system** that combines:

* Machine Learning
* Deep Learning (LSTM)
* Ensemble Learning
* Technical Indicators

to predict stock price movements and simulate real-world trading strategies.

---

## 🎯 Problem Statement

To build a robust system that predicts whether a stock price will **increase or decrease** using historical market data and advanced learning techniques, while also evaluating its effectiveness through **portfolio simulation**.

---

## ⚡ Key Features

✔ Comprehensive Technical Indicators

* RSI, MACD, Bollinger Bands
* Moving Averages (SMA, EMA)
* Momentum & Volatility indicators

✔ Machine Learning Models

* Random Forest
* Gradient Boosting
* XGBoost
* LightGBM
* CatBoost

✔ Deep Learning

* LSTM (Long Short-Term Memory) for time-series modeling

✔ Ensemble Learning

* Voting Ensemble
* Weighted Voting
* Meta-model (stacking approach)

✔ Portfolio Simulation

* Buy/Sell signal generation
* Strategy-based trading simulation
* Return and risk analysis

---
