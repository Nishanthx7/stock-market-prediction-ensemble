# 📈 Stock Market Prediction using Ensemble Learning

## 🚀 Overview

This project builds an advanced stock prediction system using:

* Machine Learning
* Deep Learning (LSTM)
* Ensemble Learning
* Technical Indicators

---

## ⚡ Features

✔ Technical Indicators (RSI, MACD, Bollinger Bands)
✔ ML Models (Random Forest, XGBoost, LightGBM)
✔ Deep Learning (LSTM)
✔ Ensemble Models (Voting, Meta-learning)
✔ Portfolio Simulation

---

## 📊 Output

* Model Accuracy & F1 Scores
* Confusion Matrices
* Trading Signals (Buy/Sell)
* Portfolio Performance

---

## ▶ Run

```bash
python stock_prediction_system.py
```

---

## 🧠 Tech Stack

* Python
* Scikit-learn
* TensorFlow
* XGBoost / LightGBM / CatBoost
